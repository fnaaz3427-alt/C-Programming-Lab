# Assignment 10 - LeetCode Array Challenge

## Objective

To practice array operations in C programming.

## Problem Statement

Create a program to find the largest, second largest, smallest, second smallest elements, reverse the array, remove duplicate elements, count the frequency of elements, perform searching, and sort the array.

## Algorithm

1. Read the array elements.
2. Sort the array.
3. Find largest, second largest, smallest, and second smallest.
4. Reverse the array.
5. Remove duplicate elements.
6. Count frequency of each element.
7. Search an element.
8. Display the results.

## Time Complexity

- Sorting: O(n²)
- Searching: O(n)
- Frequency Count: O(n²)

## Space Complexity

O(1)

## Sample Input

```
7
5 2 8 5 3 8 1
5
```

## Sample Output

```
Sorted Array:
1 2 3 5 5 8 8

Smallest = 1
Second Smallest = 2
Largest = 8
Second Largest = 8

Reversed Array:
8 8 5 5 3 2 1

Array Without Duplicates:
1 2 3 5 8

Frequency of Elements:
1 = 1
2 = 1
3 = 1
5 = 2
8 = 2

Element Found.
```
