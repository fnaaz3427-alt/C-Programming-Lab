# Assignment 03 - Pointer Playground

## Objective

To understand the basic concepts of pointers in C programming.

## Problem Statement

Write a C program to demonstrate pointer concepts such as displaying memory addresses, dereferencing pointers, swapping two numbers using pointers, traversing an array using pointers, and pointer arithmetic.

## Algorithm

1. Declare an integer variable and a pointer.
2. Store the address of the variable in the pointer.
3. Display the value and address of the variable.
4. Swap two numbers using a function with pointers.
5. Read 5 array elements.
6. Display the array using a pointer.
7. Exit the program.

## Time Complexity

- Display Address: O(1)
- Swap: O(1)
- Array Traversal: O(n)

## Space Complexity

O(1)

## Sample Input

```
10
20
1
2
3
4
5
```

## Sample Output

```
Pointer Playground

Value of num = 10
Address of num = 0x...
Value using pointer = 10

Enter first number: 10
Enter second number: 20
Before Swap: 10 20
After Swap: 20 10

Enter 5 array elements:
1
2
3
4
5

Array Elements:
1 2 3 4 5
```

## Learning Outcome

- Learned pointer basics.
- Learned how to access values using pointers.
- Learned swapping using pointers.
- Learned array traversal using pointers.
