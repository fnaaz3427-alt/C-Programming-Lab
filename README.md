# Assignment 01 - Student Marks Analyzer

## Objective
The objective of this program is to enter the marks of 5 students and find the highest marks, lowest marks, total marks, and average marks.

## Problem Statement
Write a C program to take marks of 5 students and display:
- Highest marks
- Lowest marks
- Total marks
- Average marks

## Algorithm
1. Start the program.
2. Enter marks of 5 students.
3. Store the marks in an array.
4. Find the highest and lowest marks.
5. Calculate the total marks.
6. Calculate the average marks.
7. Display all the results.
8. End the program.

## Time Complexity
O(n)

## Space Complexity
O(n)

## Sample Input
80
75
90
65
85

## Sample Output
Highest Marks = 90
Lowest Marks = 65
Total Marks = 395
Average Marks = 79.00
