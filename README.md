# Assignment 04 - Student Record Management System

## Objective

To learn how to use structures, arrays, and functions in C programming.

## Problem Statement

Create a Student Management System using structures. The program should add, display, search, update, and delete student records.

## Algorithm

1. Create a structure to store student details.
2. Display the menu.
3. Add student records.
4. Display all student records.
5. Search a student by ID.
6. Update student details.
7. Delete a student record.
8. Exit the program.

## Time Complexity

- Add Student: O(1)
- Display Students: O(n)
- Search Student: O(n)
- Update Student: O(n)
- Delete Student: O(n)

## Space Complexity

O(n)

## Sample Input

```
1
101
Rahul
20

1
102
Amit
21

2
```

## Sample Output

```
Student Added Successfully!

Student Added Successfully!

Student Records

ID: 101
Name: Rahul
Age: 20

ID: 102
Name: Amit
Age: 21
```

## Learning Outcome

- Learned how to use structures.
- Learned array of structures.
- Learned searching and updating records.
- Learned deleting records.
- Improved understanding of menu-driven programs.
