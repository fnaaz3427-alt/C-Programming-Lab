# Assignment 06 - Inventory Management System

## Objective

To learn structures, arrays, searching, sorting, and inventory management in C.

## Problem Statement

Create an Inventory Management System to store Product ID, Name, Price, and Quantity. The program should add, display, search, update, sort products by price, and calculate the total inventory value.

## Algorithm

1. Create a structure for product details.
2. Add product records.
3. Display all products.
4. Search a product using Product ID.
5. Update product details.
6. Sort products by price.
7. Calculate total inventory value.
8. Exit the program.

## Time Complexity

- Add Product: O(1)
- Display Products: O(n)
- Search Product: O(n)
- Update Product: O(n)
- Sort Products: O(n²)
- Total Inventory Value: O(n)

## Space Complexity

O(n)
