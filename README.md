# Assignment 04 - Student Record Management System

## Objective
To learn how to use structures and arrays in C for managing student records.

## Problem Statement
Create a menu-driven program that allows the user to add, display, search, update, and delete student records using an array of structures.

## Algorithm

1. Create a structure for student details.
2. Display the menu.
3. Add student records.
4. Display all records.
5. Search a student using ID.
6. Update student details.
7. Delete a student record.
8. Repeat until Exit is selected.

## Time Complexity

- Add: O(1)
- Display: O(n)
- Search: O(n)
- Update: O(n)
- Delete: O(n)

## Space Complexity

O(n)

## Sample Input

```
1
101
Rahul
20

1
102
Amit
21

2
```

## Sample Output

```
Student Added Successfully!

Student Records:

```
