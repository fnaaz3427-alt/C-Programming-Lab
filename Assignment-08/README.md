# Assignment 08 - Employee Payroll System

## Objective

To learn structures, salary calculations, and sorting in C.

## Problem Statement

Create an Employee Payroll System that accepts employee details and calculates HRA, DA, Tax, Gross Salary, and Net Salary. Display salary reports and sort employees based on salary.

## Algorithm

1. Create a structure for employee details.
2. Enter employee information.
3. Calculate HRA, DA, Tax, Gross Salary, and Net Salary.
4. Display salary details.
5. Sort employees by Net Salary.
6. Exit the program.

## Time Complexity

- Add Employee: O(1)
- Display Employees: O(n)
- Sort Employees: O(n²)

## Space Complexity

O(n)
