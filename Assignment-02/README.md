# Assignment 02 - String Utility Toolkit

## Objective
To learn basic string operations in C using functions and loops.

## Problem Statement
Create a menu-driven program to perform different string operations such as finding length, reversing a string, checking palindrome, counting vowels, digits and spaces, and displaying character frequency.

## Algorithm

1. Read a string from the user.
2. Display the menu.
3. Perform the selected operation.
4. Repeat until the user chooses Exit.

## Time Complexity

- Length: O(n)
- Reverse: O(n)
- Palindrome: O(n)
- Count: O(n)
- Frequency: O(n²)

## Space Complexity

O(1)

## Sample Input

```
madam123
```

## Sample Output

```
Length = 8
Reversed String = 321madam
String is Not Palindrome
Vowels = 2
Digits = 3
Spaces = 0
```

## Learning Outcome

- Functions
- Strings
- Loops
- Arrays
- Character processing
