# Assignment 07 - Hospital Patient Management System

## Objective

To learn structures, arrays, and functions in C programming.

## Problem Statement

Create a Hospital Patient Management System to register patients, display records, search by Patient ID, update patient information, and delete patient records.

## Algorithm

1. Create a structure for patient details.
2. Display the menu.
3. Add patient records.
4. Display all patient records.
5. Search a patient by ID.
6. Update patient details.
7. Delete a patient record.
8. Exit the program.

## Time Complexity

- Add Patient: O(1)
- Display Patients: O(n)
- Search Patient: O(n)
- Update Patient: O(n)
- Delete Patient: O(n)

## Space Complexity

O(n)
